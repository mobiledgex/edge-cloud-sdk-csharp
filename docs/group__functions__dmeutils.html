string DistributedMatchEngine.MatchingEngine.GenerateDmeHostAddress ( )

GenerateDmeHostAddress This will generate the dme host name based on GetMccMnc() -> "mcc-mnc.dme.mobiledgex.net". If GetMccMnc fails or returns null, this will return a fallback dme host: "wifi.dme.mobiledgex.net"(this is the EU + TDG DME). This function is used by any DME APIs calls where no host and port overloads are provided.

string DistributedMatchEngine.MatchingEngine.GetCarrierName ( )

Returns the carrier's mcc+mnc which is mapped to a carrier in the backend (ie. 26201 -> TDG). MCC stands for Mobile Country Code and MNC stands for Mobile Network Code. If useWifiOnly or cellular is off + wifi is up, this will return """". Empty string carrierName is the alias for any, which will search all carriers for application instances.
